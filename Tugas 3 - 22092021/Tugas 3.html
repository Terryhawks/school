How To Install SketchUp

  1. Check the device requirements
    2. Below you'll find the minimum specs needed to run SketchUp. So, please check if your device meets the requirements
    • CPU: 1GHz or faster
    • RAM: 1GB for 32-bit Win10 or 2GB for 64-bit Win10
    • Storage: 32GB of Space or more
    • GPU: DirectX 9 Compatible or later. With WDDM 1.0 or higher Driver Compatibility
    • Display: 800x600 resolution or higher

Follow these steps to install SketchUp

  1. Download the software.
    1. Follow this Link to download from the official site
    2. Click Try SketchUp
    3. Choose for Personal Project
      1. Extract to a Folder using WinRar or 7Zip
        • Right click - Choose the Extract to ... Option
        • Open The Folder
      2. Run The Executable (double click or click + Enter)
